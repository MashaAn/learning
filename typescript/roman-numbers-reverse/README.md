# Roman Numbers Reverse way

Implemented by TDD with the help of the great people of @SCC_at_DATEV 
and friends at a Coding Dojo.

## What's hot?

* uses mocha/chai
* written in TypeScript
* TSLint compliant

## How to run

```
npm install
npm test
```

## Some test cases

### addition rules

* 1 -> I
* 2 -> II
* 3 -> III
* 5 -> V
* 6 -> VI

### subtraction rules

* 4 -> IV
* 9 -> IX
